# Console-Based Restaurant Billing System (Python)

## Overview
A menu-driven console application developed in Python that allows users to place food orders, specify quantities, and generate an itemized bill with GST and discount logic.

## Key Features
- Unlimited item ordering
- Quantity-based billing
- Case-insensitive input handling
- Input validation and error handling
- GST and discount calculation
- Itemized receipt generation with timestamp
- Modular and clean code structure
